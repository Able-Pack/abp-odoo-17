Sale Group Invoice

Description

This module is configured to create a single invoice for several sales orders for the same customer. It makes billing and record-keeping much easier and more efficient.

Functionality

After installing the Sale Group Invoice application, the feature will be enabled.

Go to Sales - - > Orders - - > Orders.

Select sales orders to group into a single invoice by clicking on the “Create Invoices” option under the Action button. You can select multiple customers at once, and each will receive a separate invoice.

A wizard will appear with the following options:  

1: Create summary invoice for selected orders only - This option will generate invoices for only the selected sales orders.  

2:  Create summary invoice for all billable orders - This option will create invoices for all invoiceable sales orders for the selected customers.
